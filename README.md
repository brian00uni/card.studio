# Card Studio

가성비 크루의 여행 위급정보 릴스 카드 7장을 조사하고, 검수하고, PNG 발행본으로 확정하는 웹 프로젝트입니다.

## 운영 목표

1. 매일 오전 6시에 다음 도시의 조사·HTML 생성을 시작합니다.
2. 오전 7시 이전에 검수 화면과 QA 보고서를 준비합니다.
3. 사용자가 정보, 문구, 이미지 출처와 레이아웃을 검수합니다.
4. 승인된 결과만 1080×1920 PNG 7장으로 생성합니다.

## 서비스 역할

- Vercel: Next.js 웹앱 배포, 예약 작업, 서버 API
- Supabase: 사용자, 도시 큐, 카드 데이터, 프롬프트 버전, 검수 기록, 이미지 파일
- Hugging Face: 선택적인 이미지 분석·분류. 실제 약품 패키지 생성에는 사용하지 않음
- OpenAI API: 조사 결과 구조화, 원고 초안, 출처 및 QA 보조

ChatGPT 구독이나 로그인 정보는 OpenAI API 키가 아닙니다. API 기능을 연결하려면 OpenAI Platform에서 별도 API 키를 발급하고 Vercel 서버 환경변수 `OPENAI_API_KEY`로 등록해야 합니다. 키를 브라우저 코드나 Git 저장소에 넣지 마세요.

## 현재 단계

현재 화면은 외부 키 없이 실행되는 로컬 프로토타입입니다. 카드 미리보기, 프롬프트 편집, 출처 레지스트리, QA 및 PNG 승인 흐름을 확인할 수 있습니다. 프롬프트는 브라우저 로컬 저장소에 임시 저장됩니다.

다음 단계에서 Supabase 스키마와 로그인, 실제 카드 데이터 연결, Vercel Cron, 서버 측 OpenAI 호출, 이미지 다운로드 및 PNG 렌더링을 차례로 연결합니다.
