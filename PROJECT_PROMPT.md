# Card Studio 기본 프로젝트 프롬프트

## 목표

한국인 해외여행자가 현지에서 갑자기 아프거나 다쳤을 때 바로 사용할 수 있는 도시별 릴스 카드 7장을 만든다. 매일 오전 7시 전에 HTML 검수본을 준비하고, 사람의 승인 후에만 PNG 발행본을 생성한다.

## 자동 실행

1. 제작 대기열에서 순서가 가장 빠른 도시 하나를 선택한다.
2. 외교부, 현지 대한민국 공관, 현지 정부, 병원 공식 사이트, 제조사 공식 사이트 순으로 조사한다.
3. 전화번호, 주소, 운영시간, 지원 언어, 확인 날짜와 원문 URL을 저장한다.
4. 검색 이미지는 공식 출처와 사용 조건을 확인하고 프로젝트 저장소에 사본을 저장한다.
5. 실제 약품 패키지가 없으면 AI로 생성하지 않고 자산 필요 상태로 둔다.
6. 카드 데이터와 HTML 7장, 출처, 이미지 출처, 캡션, QA 보고서를 생성한다.
7. 샘플 문구, 불명확한 사실, 미승인 이미지, 텍스트 잘림이 있으면 검수 준비 상태로 바꾸지 않는다.
8. 사용자가 웹 검수판에서 승인한 뒤에만 PNG 7장을 생성한다.

## 발행 금지 조건

- 공식 출처와 일치하지 않는 긴급전화
- 확인 근거가 없는 24시간 또는 한국어 가능 표기
- 출처·사용 조건이 기록되지 않은 이미지
- AI로 생성한 가짜 약품 패키지
- 샘플 번호, placeholder, 잘린 문구
- 사람 검수를 통과하지 않은 현지어 문장
